# 📸 Photo Album Web Application

## 📌 Project Overview
Photo Album is a full stack web application developed for photographers to efficiently manage and share photos from events such as weddings, parties, and other occasions. It allows photographers to upload images and clients to securely access, view, and select their preferred photos.

---

## 🚀 Features
- 🔐 User Authentication (Login & Signup)
- 📤 Upload and manage photos
- 🖼️ View photos in organized albums
- ❤️ Select/favorite photos for final delivery
- 🌙 Dark Mode for better user experience
- ⚡ Dynamic content loading using AJAX
- 👥 Separate access for photographers and clients

---

## 🛠️ Technologies Used

### Frontend:
- HTML
- CSS
- JavaScript
- AJAX

### Backend:
- Java Servlets
- JSP (Java Server Pages)

---

## ⚙️ How It Works
1. Users create an account or login securely
2. Photographer uploads event photos
3. Photos are displayed dynamically using AJAX
4. Clients access their event album
5. Clients can view and select their favorite images

---

## 📂 Project Structure
- `src/` → Java Servlets (backend logic)
- `webapp/` → HTML, CSS, JS, JSP files
- `WEB-INF/` → Configuration (web.xml)

---

## ▶️ How to Run
1. Clone the repository:https://github.com/khushwant-Singh-Rajawat/Photo_Album
2. Open in IntelliJ IDEA
3. Configure Apache Tomcat Server
4. Deploy and run the project
